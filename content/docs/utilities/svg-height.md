---
title: SVG Height
date: 2018-07-20T12:59:15-04:00
---

Use these classes to change the height of an svg used for an icon.

<div class="mt-3 mb-4">
{{< highlight scss >}}
.icon--svg {
  height: 2rem;
}

.icon--svg-small {
  height: 1rem;
}

.icon--svg-large {
  height: 3rem;
}

.icon--svg-xlarge {
  height: 6.5rem;
}
{{< /highlight >}}
</div>
