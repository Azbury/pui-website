Use these classes to change the height of an svg used for an icon.

| Class Name         | Height   |
| ------------------ | -------- |
| `icon--svg`        | `2rem`   |
| `icon--svg-small`  | `1rem`   |
| `icon--svg-large`  | `3rem`   |
| `icon--svg-xlarge` | `6.5rem` |
