Use these classes to change the height of an svg used for an icon.

| Class Name         | Height   |
| ------------------ | -------- |
| `icon--svg`        | `2rem`   |
| `icon--svg-sm`  | `1rem`   |
| `icon--svg-lg`  | `3rem`   |
| `icon--svg-xlg` | `6.5rem` |
