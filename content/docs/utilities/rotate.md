Utility classes to rotate an element.

| Class Name   | Transform        |
| ------------ | ---------------- |
| `rotate-45`  | `rotate(45deg)`  |
| `rotate-90`  | `rotate(90deg)`  |
| `rotate-180` | `rotate(180deg)` |
