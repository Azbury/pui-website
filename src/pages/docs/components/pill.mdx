## [Pill](#pill)

A rounded, pill-shaped badge. A pill can support color by adding a text or background modifier.

Pills have no hover state.

<Codedemo>
<div class="pill">Active</div>
<div class="pill text--white background--navy">Active</div>
<div class="pill text--white background--skyblue">Active</div>
<div class="pill text--white background--salmon">Active</div>
</Codedemo>

<Codemarkup>
```html
<div class="pill">
  Active
</div>
<div class="pill text--white background--navy">
  Active
</div>
<div class="pill text--white background--skyblue">
  Active
</div>
<div class="pill text--white background--salmon">
  Active
</div>
```
</Codemarkup>

## [Circle](#circle)

Pills can be circular.

<Codedemo>
<div class="pill pill--circle">33</div>
<div class="pill pill--circle text--white background--salmon">33</div>
</Codedemo>

<Codemarkup>
```html
<div class="pill pill--circle">
  <!-- Optional content here! -->
</div>
<div class="pill pill--circle text--white background--salmon">
  <!-- Optional content here! -->
</div>
```
</Codemarkup>
