## [Border](#border)

When set, all borders default to a `1px` width which can be offset using `border--width-{ i }`.

### [Specific Corners](#specific-corners)

You can specify a border-radius on a certain corner of an element. Just be sure to add the corner you want. It looks like this:
`rounded-tr--2`. This would add `border-top-right-radius: .5rem`.
