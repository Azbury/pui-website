</Classes>

<Codedemo>
<div class="block-container">
    <div class="block laptop-up-3">
        <div class="card">
            <div class="card__content text-underline--hover">

            </div>
        </div>
    </div>
</div>
</Codedemo>

<Callout header="Underline Tip" >
  <p>The color of the underline will match the color of text hover utility class. If you have `text-hover--navy text-underline--hover` on an element, both text color and underline color will be navy.</p>
</Callout>
