## [Position](#position)

To postion an element absolutely, set `pos-rel` on the parent element and `pos-abs` to child that you'd like to position absolute. Our position utilities are meant to be used (mostly) with pinning.

<Classes result="true">
<Classesrow classname="pos-rel" result="Position relative to the elements normal position."/>
<Classesrow classname="pos-abs" result="Position an element absolutely."/>
<Classesrow classname="pos-fix" result="Set the fixed position of an element."/>
<Classesrow classname="pos-stick" result="Make it sticky. For use with table headers."/>
</Classes>
